Decision Tree Regressor with MAE

This project uses Scikit-learnâ€™s Decision Tree Regressor to predict house prices and calculates the Mean Absolute Error (MAE) to evaluate model performance. The script tests multiple max_leaf_nodes values and identifies the optimal number for the best prediction accuracy.

Overview

  • Reads a dataset from a CSV file containing house prices and features.
  • Trains a Decision Tree Regressor with varying max_leaf_nodes.
  • Calculates the MAE for each model and identifies the best value.

Key Functionality

  • max_mae(): Trains the model with a given max_leaf_nodes value and returns the MAE.
  • Splits the data into training and testing sets using train_test_split.
  • Loops over several max_leaf_nodes values to find the optimal model configuration.
