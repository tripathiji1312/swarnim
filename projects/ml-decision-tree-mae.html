Decision Tree Regressor with Mean Absolute Error (MAE) Calculation

This Python script uses a Decision Tree Regressor from the Scikit-learn library to predict house prices. It calculates the Mean Absolute Error (MAE) to evaluate the model's prediction accuracy and determines the best number of max_leaf_nodes based on the lowest MAE value.

Script Overview:

This script does the following:

  • Reads a dataset from a CSV file containing house prices and features.
  • Trains a Decision Tree Regressor using different values for max_leaf_nodes.
  • Calculates the Mean Absolute Error (MAE) for each max_leaf_nodes value.
  • Displays the best max_leaf_nodes value (i.e., the one that gives the lowest MAE).
  • Allows you to manually check the MAE for a given max_leaf_nodes value.

Key Functionality

  • max_mae(max_leaf_nodes, trainx, testx, trainy, testy): Trains the Decision Tree Regressor with the specified max_leaf_nodes and returns the MAE for that model.
  • train_test_split: Splits the dataset into training and testing sets.
  • Looping over max_leaf_nodes values: Trains the model for multiple values of max_leaf_nodes and calculates the MAE for each one.

Output

The script outputs:

  • The MAE for each tested value of max_leaf_nodes.
  • The minimum MAE and the corresponding best max_leaf_nodes value.
  • The MAE for any user-selected max_leaf_nodes value within the testing range.
