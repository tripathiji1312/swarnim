SecurePass: Your Encrypted Password Manager

SecurePass is a command-line based password manager built with Python. It securely encrypts and stores your passwords so that only you have access to them.

Features

  • Encryption: Uses AES-256 encryption to secure your passwords.
  • Master Password: Protect your vault with a strong master password.
  • Organization: Manage entries for various websites, usernames, and passwords in an organized manner.
  • Command-Line Interface: Interact via a user-friendly CLI for managing your credentials.
